## 🌑 About

An immersive, cinematic gaming website dedicated to **Prince of Persia: Warrior Within** (2004). Designed with a dark fantasy aesthetic featuring real game imagery, gameplay videos, 3D particle effects, and smooth scroll animations — all in a single HTML file.

> *"You cannot change your fate. No man can."* — The Old Man

---

## ✨ Features

### 🎮 Core Sections
- **Cinematic Hero** — Full-screen gameplay background with Ken Burns zoom effect
- **Story Showcase** — CSS Grid layout with official cover art and lore
- **The Dahaka** — Full-width parallax section with stylized Dahaka chase artwork
- **Gameplay Videos** — Local MP4 video player with tab switcher (Combat System + Dahaka Chase)
- **War Room** — Interactive quest tracker + screenshot gallery with lightbox
- **Enlist Form** — Validated recruitment form with success modal

### 🌌 Animations & Effects
- **Three.js 3D Particle Storm** — 700 gold & crimson sand particles + blood mist, reactive to mouse movement
- **Scroll Reveal Animations** — 4 types (fade-up, slide-left, slide-right, scale-up) via IntersectionObserver
- **Parallax Scrolling** — Depth effect on the Dahaka section
- **Micro-interactions** — Hover effects, glowing borders, floating animations, card transitions

### 🛠️ Technical Highlights
- **Single-file architecture** — HTML + CSS + JS all in one file
- **CSS Grid & Flexbox** — Responsive layouts throughout
- **Form Validation** — Real-time email & name validation with visual feedback
- **Dynamic DOM** — Quest tracker & gallery with add/remove/complete functionality
- **No frameworks** — Pure vanilla HTML, CSS, and JavaScript

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/arn99vvojash/The-Prince.git
   ```

2. **Open in browser**
   ```
   Open IntershipTask-2.html in any modern browser
   ```

3. **Optional — Add local media**  
   For the full experience with gameplay videos and images, add the following files to the project folder:
   - Game screenshots (`.jpg`)
   - Gameplay video clips (`.mp4`)
